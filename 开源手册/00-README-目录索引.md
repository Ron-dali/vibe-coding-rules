## 📦 你的资产（V2.2）

**6 个 Skill，完整流水线：**

| # | Skill | 角色 | 受众 |
|:--:|:--|:--|:--|
| 1 | `pipeline-init` | 对话式项目初始化 | 非程序员友好 |
| 2 | 编程四大纪律 (karpathy-principles) | 改前自查：先思考→做减法→精准改→能跑 | 100% 通用 |
| 3 | `safe-terminal-executor` | 终端命令安全封装 | 100% 通用 |
| 4 | `self-check` | 24 条纪律自检 + 信任分级自生长 | 可配置 |
| 5 | `web-testing` | Playwright 自动化测试 | 可配置 |
| 6 | `changelog` | 变更日志 + 生长检测 + 防膨胀 | 可配置 |

**3 大差异化卖点：**
1. **自生长规则系统** — Tier 0→1→2 信任分级，置信度自动晋升/降级/退役，现行生态中无同类产品
2. **24 条实战纪律** — 来自 3 个月 TBP 项目的真实 bug 沉淀，每一条都有血有泪
3. **双语+非程序员友好** — README 中英双语，pipeline-init 对话式初始化，零门槛
